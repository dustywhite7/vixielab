README for Nonlinear Data Analysis Project 1

Group: Greg Astill & Ethan Spangler

All python code should update to your working directory to pull in the data contained in this folder. 

All code here uses e-tongue data to predict wine type.

SVMCrossValidation.py uses k-fold cross validation and outputs prediction accuracy.

SVMGraph.py plots the SVM classifications of data points for the first two flavor characteristics.

AndrewsCurve.py plots an Andrews Curve of data point characteristics.
